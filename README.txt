					Οδηγίες Χρήσης Συστήματος 

1. Στο αρχείο config.txt δηλώνουμε τον αριθμό των νημάτων (THREADS) που θέλουμε να δημιουργεί το 
πρόγραμμα και τον αριθμό του port του server (οι default τιμές είναι THREADS=40 και PORT=8080)

2. Αρχικά ανοίγουμε το terminal και εκτελούμε την εντολή make για να μεταγλωτίσουμε το πρόγραμμα

3. Για να τρέξουμε το πρόγραμμα server.c εκτελούμε την εντολή ./server 

4. Στη συνέχεια o server περιμένει κάποια αίτηση για να συνεχίσει την εκτέλεσή του. Για να δώσουμε
κάποια αίτηση ανοίγουμε καινούριο terminal και τρέχουμε την εντολή telnet localhost 8080, όπου 8080 είναι
το port του server, και εισάγουμε την αίτηση (GET, HEAD ή DELETE) ή διαφορετικά εισάγουμε ένα URL της
μορφής http://localhost:8080/<path> σε οποιονδήποτε browser

5. Ο server παίρνει τις αιτήσεις και τις εξυπηρετεί δίνοντας πίσω απαντήσεις

6. Ο client που στέλνει τις αιτήσεις στην περίπτωση που είναι μέσω του telnet για να τερματίσει το 
connection με τον server πρέπει να περιέχει στην αίτηση του την γραμμή Connection: close

7. Σε περίπτωση που κάποια αίτηση δεν μπορεί να εξυπηρετηθεί ο server δίνει το κατάλληλο μήνυμα. 
Δηλαδή, αν το αρχείο που ζητά δεν υπάρχει ή αν η αίτηση δεν είναι GET, HEAD ή DELETE

* Σημείωση 1: Αν θέλουμε να κάνουμε debug το πρόγραμμα εκτελούμε την εντολή make debug σε κάποιο terminal
και μετά εκτελούμε την εντολή ./tests

* Σημείωση 2: Αν θέλουμε να διαγράψουμε τα .o και τα executable αρχεία που δημιουργήθηκαν εκτελούμε την
εντολή make clean 

*** ΣΗΜΑΝΤΙΚΟ: Τα αρχεία που επιστρέφει ο server πρέπει να υπάρχουν στον ίδιο φάκελο με το αρχείο server.c
