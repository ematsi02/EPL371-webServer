					Οδηγίες Χρήσης Συστήματος 

1. Αρχικά ανοίγουμε το terminal και τρέχουμε το αρχείο makefile με την εντολή make για να μεταγλωτίσουμε
το πρόγραμμα

2. Για να τρέξουμε το πρόγραμμα server.c το οποίο περιέχει την συνάρτηση main εκτελούμε την εντολή
./server 

3. Στη συνέχεια o server περιμένει κάποια αίτηση για να συνεχίσει την εκτέλεσή του. Για να δώσουμε
κάποια αίτηση ανοίγουμε καινούριο terminal και τρέχουμε την εντολή telnet localhost 8080, όπου 8080 είναι
το port του server, και εισάγουμε την αίτηση (GET, HEAD ή DELETE) ή διαφορετικά εισάγουμε ένα URL της
μορφής http://localhost:8080/<path> σε οποιονδήποτε browser

4. Ο server παίρνει τις αιτήσεις και τις εξυπηρετεί δίνοντας πίσω απαντήσεις

5. Ο client που στέλνει τις αιτήσεις στην περίπτωση που είναι μέσω του telnet για να τερματίσει το 
connection με τον server πρέπει να περιέχει στην αίτηση του την γραμμή Connection: close

6. Σε περίπτωση που κάποια αίτηση δεν μπορεί να εξυπηρετηθεί ο server δίνει το κατάλληλο μήνυμα. 
Δηλαδή, αν το αρχείο που ζητά δεν υπάρχει ή αν η αίτηση δεν είναι GET, HEAD ή DELETE.
